# vim 常用快捷键记录 #
1. 替换与全局替换
   ````
    :[range]s/pattern/string/[c,e,g,i]
    
    range表示替换的范围
          ：s默认当前行
          ：%s 全文
          ：'<,'>s 选区范围内
          ：1,10s 1-10行
          ：.,+10s 从当前行开始的接下来10行
          
    patter 被替换时的正则表达式
    string 替换的字符
    c      confirm，每次替换前询问确定
    e      不显示error
    g      globe,不询问，在选择的范围内全局替换
    i      ignore 不分大小写
   ````
