# vim 常用快捷键记录 #
1. 替换与全局替换
   ````
    :[range]s/pattern/string/[c,e,g,i]
    
    range表示替换的范围
          ：s默认当前行
          ：%s 全文
          ：'<,'>s 选区范围内
          ：1,10s 1-10行
          ：.,+10s 从当前行开始的接下来10行
          
    patter 被替换时的正则表达式
    string 替换的字符
    c      confirm，每次替换前询问确定
    e      不显示error
    g      globe,不询问，在选择的范围内全局替换
    i      ignore 不分大小写
   ````
2. 移动 
    ````
            gg: 跳转到文件开头
       shift+g: 跳转到文件末尾
       行数+g:   跳转到指定行数 
          10j:  向下移动10行   
          w:    移动光标到下一个单词的词首
          b:    移动光标到上一个单词的词首
          e:    移动光标到下一个单词的结尾
          ge:   移动光标到上一个单词的结尾   
          ^:    移动到行首
           $:   移动到行尾 
        ctrl+f: 向下翻页
        ctrl+b  向上翻页
          
    ````
    
3. 删除/剪切
    ````
        dd 删除当前行
       ndd 删除当前行开始的n行
       
    ````
    
4. 复制/粘贴
    ````
       x,y copy n 复制连续x行,y行 插入到n行开始的后面
        yyp       复制当前行到下一行
        p         粘贴剪切、复制的内容
        nyy       复制n行
                
    ````
